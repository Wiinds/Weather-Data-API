URL Format: http://127.0.0.1:5000/api/v1/station/date

URL Example for one date: http://127.0.0.1:5000/api/v1/10/1988-15-25

URL Example for one station all dates: http://127.0.0.1:5000/api/v1/10

URL Example for one station for one year: http://127.0.0.1:5000/api/v1/yearly/10/1978
